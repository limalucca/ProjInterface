LookAfter


O projeto se baseia em um site, onde nesse site será para vendas de roupas de inverno com uma ação beneficente para instituições de caridade ou ONGs de diferentes categorias.

O planejamento principal é utilizar as cores das roupas de inverno para diferenciar as instituições/ONGs para que cada roupa mande uma porcentagem da venda direto para diferentes organizações.

Estamos pensando em incluir no site uma forma do próprio cliente de doar para essas organizações, fazendo uma página única somente para os usuários conseguirem doar uma peça ou mais peças de roupas.

Também adicionaremos uma opção para fazer doações de roupas para animais, como uma forma de ser um site amplo e alcançar uma comunidade maior, assim teremos um diferencial dos outros projetos.
